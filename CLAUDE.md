# Project Brain

## Stack
Next.js 14, Tailwind + shadcn/ui, Supabase, Stripe, Vercel

## Commands
npm run dev | npm run build | npm test | npm run lint

## Conventions
- TypeScript strict, no `any`
- Zustand for state
- Dark mode first
- Auto-commit after every change
